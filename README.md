# homebrew-subtui
## Installation

You can install SubTUI directly with a single command:
```bash
brew install MattiaPun/subtui/subtui
```

Alternatively, you can "tap" the repository first to track updates more easily:
```bash
brew tap MattiaPun/subtui
brew install subtui
```

## Updating

To update SubTUI to the latest version:
```bash
brew update
brew upgrade subtui
```
